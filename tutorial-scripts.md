## Technical Production Notes

### Recommended Screen Recording Setup:
- **Resolution:** 1080p (1920x1080) minimum
- **Frame Rate:** 30 FPS
- **Audio:** Clear voiceover with screen recording software
- **Cursor:** Enable cursor highlighting for better visibility

### Website Recommendations for Demos:
- **Safe for recording:** All mentioned sites are publicly accessible
- **No login required:** Use public pages when possible
- **Fast loading:** Choose responsive, well-designed sites
- **Relevant content:** Match the use case being demonstrated

### Editing Guidelines:
- **Smooth transitions** between different sections
- **Zoom highlights** on important UI elements
- **Consistent pacing** - not too fast, not too slow
- **Clear audio** with good microphone quality
- **Call-to-action overlays** with website URL

### Captions & Accessibility:
- **Auto-generated captions** as baseline
- **Manual caption review** for accuracy
- **Visual descriptions** of on-screen actions
- **Clear contrast** for any text overlays

### Final Output Specifications:
- **Format:** MP4 with H.264 encoding
- **Quality:** High quality for website embedding
- **File size:** Optimized for web delivery
- **Thumbnails:** Custom thumbnails for each video showing key features
